AORTIX – AI Cardiovascular Digital Twin System

AORTIX is a multi-agent AI-powered cardiovascular digital twin platform that analyzes patient vital data and generates intelligent treatment recommendations using deep learning, clinical safety rules, and generative AI.

The system processes physiological inputs (heart rate, blood pressure, SPO₂, ECG risk, pulmonary pressure, medication status) and produces:

 Organ-level heart risk prediction using LSTM (PyTorch)
 Treatment effectiveness simulation using MLP model
 Clinical safety validation with Risk Severity Index (RSI)
 Confidence-based treatment decision engine
 Automated clinical summary using Phi-3 Mini (Ollama GenAI)

Built with a FastAPI multi-agent architecture, AORTIX simulates a digital twin of cardiovascular health to support explainable AI-based clinical decision making.

Tech Stack: FastAPI, PyTorch (LSTM + MLP), Pandas, NumPy, Scikit-learn, Joblib, Ollama (Phi-3 Mini)
